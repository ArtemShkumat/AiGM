*   **ID Generation:** Only use the provided `id` for the Wilderness area itself.
*   **NPCs:** Do **not** include specific NPC lists or IDs. Use the `typical_occupants` field for general guidance.
*   **Connections:** Do **not** include explicit `connected_locations` properties. Its placement is defined by `parentLocationId`.
*   **JSON Format:** Ensure the output is valid JSON matching the `Wilds` model structure.
