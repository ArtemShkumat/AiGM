1. SystemNPCGeneratorInstructions	Instructs LLM to create a rich, detailed NPC.
2. ResponseInstructions	Defines output format (freeform, not JSON).
3. ExampleResponses	Guides the LLM towards creative but useful results.
4. GameSetting	Ensures NPC fits within the established world.
5. GamePreferences	Adjusts tone, complexity, and age rating.
6. TriggerInstructions
6. Trigger	Defines why this NPC is being created (e.g., quest, location, random encounter).
7. TriggeringContext	Passes relevant world data (quest details, location traits, or player request).
